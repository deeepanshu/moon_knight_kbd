# Moon Knight Keyboard

A custom split keyboard project with multiple hardware iterations and utilities.

## Project Structure

- **moon_knight_v1/** - Initial keyboard design
- **moon_knight_v2_left/** & **moon_knight_v2_right/** - Second iteration split keyboard
- **moon_knight_v3/** - Third iteration
- **moon_knight_v4/** - Latest iteration
- **firmwares/** - Keyboard firmware configurations
- **keeb_utils/** - Utility tools for the keyboard
  - **layer-display/** - Real-time layer visualization utility
